## 自然発生条件
敵は30ターンごとにポポロのいない部屋（大部屋の場合は視界外）に自然発生します。

## 自然発生の制限
トルネコ3は計算量を制限するためか、フロアのユニット数は40匹に制限されています。
```math
（ポポロ）+（仲間ユニットの数）+（敵ユニットの数）+（ガーゴイルの数）≦ 40
```

さらに、自然発生する敵ユニット数も制限されています。
```math
（自然発生により発生する敵ユニットの数）≦ 20
```

## 自然発生以外の発生要因
- スモールグールの分裂
- マドハンドの仲間呼び
- 分裂の杖
- ひとくいばこ、ミミック
- モンスターハウスの敵

## コンボへの応用
たとえば、仲間の数が10ならば、
- 敵の数の合計を29匹（=40-1-10）とすれば、分裂スモグルは発生しません
- 敵の数を合計20匹とすれば、自然発生はなくなり、最大9匹（=40-1-10-20）の分裂スモグルが発生します

## 自然発生で優先されないマス
以下のマスは自然発生で優先されないことがわかっています。  
不要なアイテムを階段部屋にばらまく等で、階段を降りる際に安全な状態を作り出すことができます。

- トゲ床
- ツルツル床
- 店のある部屋の最外周
- アイテムの置いてあるマス
- 罠のあるマス
- 他のキャラのいるマス
